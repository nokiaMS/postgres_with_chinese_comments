# 源代码目录说明
+ src - 内核代码
  + backend
    + utils
      + adt - 抽象数据类型定义，pg的内置数据类型定义在此处。
        + date.c - 定义了date,time,timetz类型相关的接口函数及支持函数。
        + timestamp.c - 定义了timestamp及interval类型相关的接口函数及支持函数。
    + parser - 解析器源代码。
    + optimizer - 优化器源代码。
    + statistics - 统计信息相关。
    + rewrite - 查询重写。
    + executor - 执行器相关。
    + catalog - 系统表相关。
    + storage - 存储相关。
    + tcop - traffic cop(流量警察，用于后端接收请求并分发请求到对应的模块进行处理。)
    + libpg - libpg相关。
    + foreign - foreign table相关。
    + jit - jit相关。
    + partitioning - 分区相关。
    + po - portable object(可移植对象)，包含各种语言的国际化支持文件，里面包含了各种语言的翻译资源。
    + tsearch - 全文检索相关。
  + bin - pg工具源代码。
    + initdb - initdb命令源代码。
    + pg_config - pg_config命令源代码。
    + pg_dump - pg_dump源代码。
    + pgbench - pgbench源代码。
    + psql - psql源代码。
  + include - 包含各种模块的头文件。
    + utils
      + date.h - date，time，timetz相关接口声明。
+ contrib - 扩展代码
