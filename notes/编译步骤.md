## 如何编译时不编译文档
在make install world之前执行如下两个命令，会在make install world的时候省略编译文档的步骤： 
（编译文档需要环境中安装一些工具，一些环境下可能不能安装这些工具或者不需要进行文档编译，此时可以去掉文档的编译步骤。）
    
    sed -i 's/world,doc/world,/' GNUmakefile
    sed -i 's/install-world,doc/install-world,/' GNUmakefile

编译后的文档位置：

    release/share/doc/html
    release/share/doc/extension
